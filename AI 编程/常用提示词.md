# AI 编程常用提示词

## git 类

```text
将当前工作区的所有修改内容都提交到 git 仓库，要求如下：
（1）将修改内容按照功能模块定位等，可以拆分为多次提交。
（2）最终要求工作区是干净的。
（3）提交消息需要按照最佳实践的中文表达。
（4）所有事项完成后，进行一次推送远程仓库操作。
```

```text
将当前工作区的修改内容进行一次 git 提交，注意使用中文的提交消息，并推送至远程仓库。
```

将以下代码规范以合适的方式添加到 `DevOps/开发规范.md` 和 `CLAUDE.md` 文件中：
（1）不允许包含未使用的 import 语句
